Raising Fighting Spirit

Jogo em C++ desenvolvido para aplicar na prática os conceitos fundamentais da Programação Orientada a Objetos.

Sobre o Projeto

O Raising Fighting Spirit é um projeto acadêmico desenvolvido em C++, que tem como objetivo demonstrar a aplicação prática dos conceitos de Programação Orientada a Objetos (POO) por meio de um jogo interativo em ambiente de terminal. No jogo, o jogador controla um personagem que busca aprimorar seu “espírito de luta”, enfrentando eventos aleatórios, desafios e decisões que influenciam diretamente sua evolução durante a jornada. A estrutura do código foi planejada com base em princípios sólidos de design de software, sendo dividida em módulos independentes e organizados em classes, como Game, Events, Interface e SpiritGenerator. Essa arquitetura modular permite um desenvolvimento mais limpo, reutilizável e de fácil manutenção. Durante o desenvolvimento, foram explorados os principais pilares da POO — abstração, encapsulamento, herança e polimorfismo — aplicados à criação de personagens, manipulação de eventos e gerenciamento da lógica de jogo. O projeto também destaca o trabalho em equipe, com cada integrante responsável por partes específicas, como lógica do jogo, sistema de eventos, interface e documentação.

Como Executar

Para compilar e executar o projeto Raising Fighting Spirit, é necessário ter um compilador C++ instalado, como o g++ (Linux) ou o MinGW (Windows). O jogo é executado diretamente no terminal, sem necessidade de interface gráfica.

Abra o terminal e acesse o diretório do projeto:


cd raising-fighting-spirit

Compile todos os arquivos-fonte com o seguinte comando:


g++ -o raising_fighting_spirit main.cpp Game.cpp Events.cpp Interface.cpp SpiritGenerator.cpp -std=c++17

Após a compilação bem-sucedida, execute o programa:


./raising_fighting_spirit
